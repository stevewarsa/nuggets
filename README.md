# Bible Nuggets
This is a Bible browsing and memory tool.  I have developed it for my own use, but it may be useful for others as well.  In this tool, you can:
1. Read the Bible (chapter at a time)
2. Randomly browse passages from the Bible
3. Search the full bible for any word or phrase including wildcards.  The search can be limited by Old/New Testament or by a particular book.  The search can search any one of 9 supported Bible translations
4. Browse Bible passages by topic (either select a topic or let the tool present you random topics for selection).
5. Practice memory verses as well as other facts you might want to remember
